# **Amazon-clone**
A fully functional front-end clone of the Amazon website, built using HTML and CSS. This project replicates the look and feel of Amazon's user interface, including the navbar, product listings, and footer.
## **Project Overview**
This project is a static front-end clone of Amazon.com, created as a practice project. The goal was to closely replicate Amazon's visual design using only HTML and CSS.
